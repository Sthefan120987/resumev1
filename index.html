Microsoft Word

Microsoft Word es un procesador de textos que permite crear documentos como trabajos escolares, informes y cartas. En secundaria, es esencial aprender a dar formato para presentar escritos de manera clara y ordenada.

Microsoft Excel

Microsoft Excel organiza datos en tablas y realiza cálculos automáticos. Es muy útil para notas, horarios o estadísticas. Su potencia está en las fórmulas y gráficas.

Microsoft PowerPoint

Microsoft PowerPoint ayuda a crear presentaciones llamativas con texto, imágenes y animaciones. Es muy útil para exponer temas en clase de forma clara.
